🍽️ Zomato Data Analysis (Python)

This project is a data analysis notebook based on the Zomato dataset using Python. The main goal of this project is to explore restaurant data and find useful insights such as customer preferences, popular restaurant types, rating distribution, and online order trends.

📌 Features / Analysis Done
Data cleaning and preprocessing
Handling missing values
Exploratory Data Analysis (EDA)
Restaurant rating distribution analysis
Online order vs offline order comparison
Most popular restaurant types and locations
Visualizations using graphs and charts
🛠️ Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
📊 Output

This project helps to understand:

which restaurant types are most popular
how ratings are distributed
how online ordering impacts restaurant performance
customer preference patterns
